Drop your photos here with these exact filenames and they'll appear automatically (the page already references them):

- `hero.jpg` — opening photo (top of page)
- `meteo.jpg` — weather visual (Avant de partir)
- `temperatures.jpg` — temperature chart (Le climat)
- `transports.jpg` — shinkansen / train photo (banner)
- `culture.jpg` — torii / street photo (banner)

Any image format works (jpg/png/webp) as long as the filename + extension matches what's in `index.html`. If you use a different extension, update the matching `<img>`/placeholder `data-slot` in `index.html` (or tell Claude to do it).

Recommended sizes: hero.jpg roughly 1000x1250px (portrait), the rest roughly 1600x900px (landscape) — but any reasonable size will work, they'll be cropped to fit.
